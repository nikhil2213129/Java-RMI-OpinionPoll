# Java-RMI-OpinionPoll
Opinion Poll Using Java RMI

# Linux WSL 
1.Clone the Repo
2.Get into the directory 
3.Compile the java file using javac *.java
4.Turn on the Server using java PollingServer
5.Move to another wsl tab 
6.Turn on Client using java PollingClient
7.Thats It. You can Poll for the count.

This is a first phase LLD for Election Polling System Using Java RMI
