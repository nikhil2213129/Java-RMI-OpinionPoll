# Java-RMI-OpinionPoll
Opinion Poll Using Java RMI

# Linux WSL 
Clone the Repo
Get into the directory 
Compile the java file using javac *.java
Turn on the Server using java PollingServer
Move to another wsl tab 
Turn on Client using java PollingClient

Thats It. You can Poll for the count.

This is a first phase LLD for Election Polling System Using Java RMI
